Proyecto 1: Listas de tareas

Descripción del proyecto

Es una aplicación web sencilla diseñada para ayudar a los usuarios a gestionar sus tareas diarias. Permite a los usuarios añadir, editar y marcar tareas como completadas, ayudándoles a mantenerse organizados y productivos. He utilizado las siguientes tecnologias.

  • Frontend: HTML5, CSS3, JavaScript
  • Almacenamiento Local: LocalStorage para guardar las tareas del usuario en el navegador

Futuras mejoras

Me he planteado añadirle diferentes funcionalidades nuevas.

  • Sincronización en la Nube: Implementar una funcionalidad que permita a los usuarios sincronizar sus tareas en la nube para acceder a ellas desde diferentes dispositivos.
  • Subtareas: Permitir a los usuarios crear subtareas dentro de una tarea principal para una organización más detallada.
  • Integración con Calendarios: Integrar la aplicación con Google Calendar, Outlook, y otros servicios de calendario para visualizar las tareas junto con otros eventos.
  • Exportación e Importación: Permitir a los usuarios exportar e importar sus listas de tareas en formatos como CSV o JSON para mayor flexibilidad.

Funcionalidades

Funcionalidades del proyecto:

  • Añadir Tareas: Los usuarios pueden añadir nuevas tareas escribiendo el nombre de la tarea en un campo de entrada y haciendo clic en un botón para agregarla a la lista.
  • Editar Tareas: Los usuarios pueden editar las tareas existentes para cambiar el nombre o la descripción.
  • Marcar como Completadas: Los usuarios pueden marcar las tareas como completadas, lo que las tacha de la lista pero las mantiene
  • Eliminar Tareas: Los usuarios pueden eliminar tareas que ya no son necesarias.
  • Filtrar Tareas: Los usuarios pueden filtrar las tareas para ver todas, solo las completadas, o solo las pendientes.
